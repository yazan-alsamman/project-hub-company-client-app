## How to Update This Changelog

When making changes:

1. Add your changes under the `[Unreleased]` section
2. Use the appropriate change type (Added, Changed, Fixed, etc.)
3. Provide a clear description of what changed
4. Link to related issues or pull requests when possible
5. When releasing a new version, move `[Unreleased]` items to a new version section

### Example Entry

```markdown
## [Unreleased]

### Added
- New feature: Task filtering by priority
- Support for project templates

### Changed
- Improved task creation UI
- Updated project dashboard layout

### Fixed
- Resolved issue with task deletion (#123)
- Fixed theme switching bug (#124)
```
